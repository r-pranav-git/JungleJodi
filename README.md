# 🌲 JungleJodi

JungleJodi is a creative full-stack web platform that reimagines modern dating applications for animals living in a forest ecosystem.  
Instead of human-centric preferences and text-based chats, JungleJodi focuses on ecological compatibility, immersive forest interactions, and playful storytelling.

---

## ✨ Key Highlights
- Ecosystem-aware matchmaking with predator–prey restrictions
- Forest-themed interactions replacing traditional chats
- Seasonal UI modes: Spring, Monsoon, and Night Forest
- Gamified experience with animations and ambient forest sounds

---

## 🛠️ Tech Stack
- React + Vite
- Tailwind CSS
- Express.js
- Zustand (state management)
- Framer Motion (animations)

---

## 🔄 Application Flow
1. Create an animal profile using ecological traits  
2. Discover compatible matches based on forest logic  
3. Interact through creative forest activities  
4. Experience dynamic seasonal themes and immersive UI
